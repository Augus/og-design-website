表單元件（Form Element）

輸入框大小（Input Size）

輸入框是網頁最基本的內容輸入元件，通常我們會根據不同的情境使用不同的大小，元件庫提供4種不同尺寸，分別是.input-xs、.input-s、.input-m、.input-l，如下方案例說明：

						
<input class="input-xs" type="text" placeholder="在這裡輸入內容">

<input class="input-s" type="text" placeholder="在這裡輸入內容">

<input class="input-m" type="text" placeholder="在這裡輸入內容">

<input class="input-l" type="text" placeholder="在這裡輸入內容">

輸入框圖示（Input with ICON）

有時候為了讓畫面更加的簡潔，某些輸入框不會搭配 Label 來展示，這時候如果只剩下 placeholder 十分的不清楚，這種情況使用輸入框圖示可能是一個不錯的解決方案之一。在元件庫中，我們可以使用.icon-left及.icon-right來達到這個需求，如下方案例說明：

輸入框狀態（Input Status）

良好設計的輸入框應該具備即時輸入狀態顯示的能力，隨著使用者輸入內容的不同，即時提供正確、錯誤、警告等狀態的提示是很重要的，在元件庫中，我們可以使用.input-positive、.input-negative、.input-warning三種 class 來達到這個需求。
